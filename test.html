üîç How It Solves the Medicine Matching Problem

Problem:

FDA database requires exact medicine names, but extracted names might be:

  • Brand names vs Generic names
  • Different spellings/abbreviations
  • Indian/International medicines not in FDA
  • Partial names from image text

Solution:

  1. Gemini Extraction: Gets brand name, generic name, active ingredients
  2. Search Variations: Creates multiple search terms
  3. FDA Multi-search: Tries different endpoints and fields
  4. Name Normalization: Removes dosage/form suffixes
  5. Common Mappings: Maps known medicine name variations
  6. OpenRouter Fallback: For non-FDA medicines
